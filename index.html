    Shroombase is a collection and documentation data website for mushrooms and fungi. Searching for fungi that have specific attributes (tree types, regions) while also having photos that clearly show identification of those attributes is difficult to do online. Although there are many awesome resources out there that classify fungi well, this site seeks to simplify and create a visually-oriented profile system for select fungi, which can be understood by hobbyists and those new to the fungi world.
